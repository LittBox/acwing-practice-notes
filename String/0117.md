# 5288.字符串 https://www.acwing.com/problem/content/5291/
题目要求：给定一个由小写字母构成的字符串 s，请你将其翻转后输出。

难度：⭐

解题代码：
```c++
#include<iostream>
#include<string>
using namespace std;

int main(){
    string s;
    cin >> s;
    int s_len = s.length();
    for(int i = s_len - 1; i >= 0; i--){
        cout<<s[i];
    }
    return 0;
}
```

# 3640.删除字符串 https://www.acwing.com/problem/content/3643/
题目要求：给你一个字符串 S，要求你将字符串中出现的所有 `gzu` 子串删除，输出删除之后的 S。

难度：⭐

**唯一注意一点：**

遍历字符串要将字符串全部遍历完，可以在循环遍历的内部加一个越界的条件判断。

解题代码：
```c++
#include<iostream>
#include<string>
using namespace std;

int main(){
    string s = "";
    cin >> s;
    int s_len = s.length();
    string s_out = "";
    // 循环条件改为i < s_len，确保遍历所有字符
    for(int i = 0; i < s_len ; ){
        // 检查i+2是否不超过字符串长度，避免越界
        if(i + 2 < s_len && s[i] == 'g' && s[i+1] == 'z' && s[i+2] == 'u'){
            i += 3;
            continue;   
        }
        s_out += s[i];
        i++;
    }
    cout << s_out;
    return 0;
}
```

# 5054.拼接字符串 https://www.acwing.com/problem/content/5057/
题目要求：对于三个由英文字母构成的字符串 A,B,C，如果将它们按某种顺序进行首尾相连拼接，可以得到一个由英文字母构成的字符串 D，那么就称 D是 A,B,C的拼接字符串。

在进行上述判断时，忽略英文字母的大小写。

难度：⭐⭐

解题思路：
* 将所有字符串进行预处理(去字符，转小写)
* 使用枚举，将所有可能的组合放入set集合
* 使用set类中的`count()`方法，判断Q字符串是否是拼接字符串
  * `count()` = 0， 输出 `WA`
  * `count()` = 1， 输出 `ACC`
  
踩坑点：
* `set`类不能使用`[]`操作符进行访问和存取操作
> `set`是关联式容器，容器内的元素没有固定的位置，默认按照升序排列，插入操作使用`insert()`方法
* `set`默认初始化没有`set<T> sname(10)`这个方法
* 容易忽略`忽略大小写`这个条件

解题代码：
```c++
//将所有的字符要进行去符号
//三个字符串有六种组合方式
//将六种组合方式放入set集合，使用count()去找，返回1，输出ACC，返回0，输出WA
#include<iostream>
#include<string>
#include<set>
#include<vector>
using namespace std;

//将去符号这个功能进行封装, 要忽略大小写
//补充操作，将
string delete_sign(string s){
    string result;
    int s_len = s.length();
    for(int i = 0; i < s_len; i++){
        if(s[i] == '-' || s[i] == '_' || s[i] == ';'){
            continue;
        }
        if(s[i] >= 'A' && s[i] <= 'Z' ){
            result += s[i] - 'A' + 'a';
            continue;
        }
        result += s[i];
    }
    return result;
}


int main (){
    vector<string> ss(3);
    for(string &s : ss){
        cin >> s;
    }
    int n;
    cin >> n;
    
    vector<string> ss_delete(3);
    for(int i = 0;i < 3; i++){
        ss_delete[i] = delete_sign(ss[i]);
    }
    
    // 关键修正：创建空set，不再指定长度
    set<string> pattern;
    pattern.insert(ss_delete[0] + ss_delete[1] + ss_delete[2]);
    pattern.insert(ss_delete[0] + ss_delete[2] + ss_delete[1]);
    pattern.insert( ss_delete[1] + ss_delete[0] + ss_delete[2]);
    pattern.insert(ss_delete[1] + ss_delete[2] + ss_delete[0]);
    pattern.insert(ss_delete[2] + ss_delete[1] + ss_delete[0]);
    pattern.insert(ss_delete[2] + ss_delete[0] + ss_delete[1]);
    
    for(int i = 0; i < n ; i++){
        string s,s_delete;
        cin >> s;
        s_delete = delete_sign(s);
        int canJoin = pattern.count(s_delete);
        if(canJoin == 1){
            cout << "ACC" <<endl;
        }else{
            cout << "WA" << endl;
        }
    }
    
    return 0;
}
```
